Configuração do Git

Nesta página, você aprenderá a fazer a configuração do Git:

  1. Acesse o Git no Windows ou terminal do Linux.

    2. Se você estiver no Linux, basta abrir o Terminal e digitar os seguintes comandos. Porém, no Windows você deve abrir o Menu de Pesquisa e digitar "Git Bash" ou acessar o Prompt de Comando.

    Para que seja configurado o nome de usuário de sua conta do Git, basta digitar o código abaixo e colocar seu nome de usuário de sua conta no GitHub:

    $ git config --global user.name “seu-nome”

    Para que seja configurado seu email para a vinculação com a sua conta do Git, basta digitar o código abaixo e colocar seu email utilizado em sua conta no GitHub:

    $ git config --global user.email “seu email”

  2. Pronto! O Git está configurado!

    3. Para checar se as configurações estão corretas, digite o comando:

    $ git config --list

    O Git possui um editor de texto incluído na instalação, mas você pode utilizar outros de sua preferência.

Lembrando:

  • $ git config
    é o comando para acessar as configurações do Git;

  • --global
    é a gravação do seu nome e email;

  • user.name
    é o seu nome de úsuario de sua conta no Git Hub;

  • user.email
    é o seu email utilizado em sua conta no Git Hub;

  • --list
    é a lista de configurações.
