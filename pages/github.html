GitHub

O GIT HUB é uma plataforma online de trabalho colaborativo por meio de respositórios, onde os usuários compartilham seus projetos e onde pessoas de todo mundo podem trabalhar paralelamente neles, pense nela com uma rede social de desenvolvedores.

Desenvolvida por Chris Wanstrath, J. Hyett, Tom Preston-Werner e Scott Chacon, o primeiro commit realizado pela empresa em 2007, em São Francisco.

A plataforma possibilita a criação de repertórios ilimitados (em vigor desde 2019), ferramentas de revisão de código, suporte a Wiki e rastreamento de erros. Porém, o GitHub possui duas versões de contas de usuário: a gratuita (todos benefícios citados anteriormente) e a paga (adicionando funções como a criação de branches), além de depender de ferramentas de terceiros para Sistemas de CI/CD.

Pense no GITHUB, como o local em que as versões feitas no GIT serão armazenadas na internet, para o compartilhamento e as alterações. Para utilizar deve-se criar uma conta, criando assim seu repositório, lugar onde seu código ficará armazenado.

A Rede Social dos Desenvolvedores

O Git é a fonte primordial do GitHub, onde os comandos através do Git serão enviados para o repositório do programador cadastrado na plataforma. O Hub significa a socialização com pessoas que possuem os mesmos interesses de projeto que você.

Simplificando a explicação pelo nome:

  • “GIT” – Programa de sistema de controle de versão utilizado
  • “HUB” – A conexão ente programadores do mundo todo para trabalhar no código
