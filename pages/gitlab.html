GIT LAB

O GIT LAB tambem é uma plataforma online de trabalho baseada em repositórios, mas diferentemente do GIT HUB, ele é Open Source, criado em 2011, pela empresa GitLab B.V

O que torna ele de grande utilização, é a chance de você poder baixa-lo e instalá-lo no seu próprio servidor, para facilitar sua atividade no dia a dia, tendo assim uma interface web para gerir seus códigos, se o usuário tem a intenção de trabalhar em seu código, só com desenvolvedores da sua própria equipe, o GIT LAB, pode ajudar a acelerar um pouco o seu fluxo de trabalho.

E ele tambem tem a hospedagem publica, basta você criar sua conta e armazenar seus códigos num repositório online.
