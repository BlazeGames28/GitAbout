Instalação do GIT no Linux

Nesta página, você aprenderá a fazer a instalação do Git no Linux por etapas:

  1. Acesse o terminal do Linux.

  2. Digite os seguintes comandos:

    • Caso a distribuição seja Debian ou Ubuntu:

      • $ sudo apt-get update
      $ sudo apt-get install git

    • Caso a distribuição seja da família HadHat (Fedora, CentOS, ...):

      • $ yum install git-core

  3. Caso precise verificar a versão do Git:

    4. $ git --version

    O sudo permite que o usuário faça alterações como outro usuário, em geral como um superusuário (root).

    O apt-get é um gerenciador de pacotes, ele permite que você faça downloads de atualizações e de novos arquivos.

Pronto! O Git foi instalado no Linux com Sucesso!

Diferente da instalação do Git no Windows, no Linux é por meio de comandos no terminal. Agora desfrute seus projetos com o Git!
